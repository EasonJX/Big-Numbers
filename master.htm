There are roughly 6 kinds of programs in this repository
19
Mytools2015: Of interest could be:
20
  CheckMnemonisc : Runs through a .rc file (generated by VC++) and find inconsistencies or missing accelerator entries
21
  CheckRefCount  : When a program is to be checked for memoryleaks, one can use the macroes TRACE_NEW, TRACE_DELETE...
22
                   (defined in Lib/MFCUtil/D3DeviceFactory) to make af log of all memory allocation/deallocation/AddRef/Release
23
  FindStrings    : If _T("strings") - literals is to be used in all files of a project, one can use this program to find "old-style"
24
                  string-literals, without the _T(...) around
25
26
MyTools2015/Lib  : All commonly used classes and functions used by many of the other project
27
Math             : Mathematical programs
28
Parsergen        :An LALR(1) parsergenerator and generator for lexical analysis used in f.ex. Lib/Math/Expression.y and .lex
29
Test and UnitTest: Unittest for lib and other test programs
30
UnixTools        : Some of the most commonly used unix-commands. (grep, cut, cat, du, nl, head, tail, uniq, ...
